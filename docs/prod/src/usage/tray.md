# 系统托盘

Ribopper 通过系统托盘集成，让您可以方便地访问剪贴板历史和应用设置。

## 托盘图标

Ribopper 启动后，会在系统托盘显示应用图标，表示应用正在后台运行。

### 图标状态

- **正常状态**: 应用正常运行
- **活动状态**: 剪贴板历史窗口打开时

## 左键点击

左键点击托盘图标可以打开剪贴板历史窗口，显示所有保存的剪贴板记录。

### 窗口功能

- 显示剪贴板历史记录列表
- 提供搜索功能
- 显示操作按钮

## 右键菜单

右键点击托盘图标会显示菜单选项，提供快速访问常用功能的方式。
