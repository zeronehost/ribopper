# 剪贴板历史
Ribopper 会自动保存您的剪贴板历史记录，让您可以随时访问之前复制的内容。

## 查看历史记录

### 打开历史记录窗口

- 左键点击系统托盘图标
- 或使用快捷键（默认可在设置中查看和修改）

### 记录列表

历史记录窗口显示所有保存的剪贴板内容，按时间倒序排列（最新的在最前面）。

### 记录类型

支持显示以下类型的剪贴板内容：

- **文本内容**: 显示文本预览
- **文件路径**: 显示文件路径和图标

## 搜索功能

使用搜索框可以快速查找特定的剪贴板记录：

1. 在搜索框中输入关键词
2. 系统会实时过滤显示匹配的记录
3. 支持文本内容的全文搜索

## 使用历史记录

### 复制到剪贴板

1. 在历史记录列表中找到需要的记录
2. 点击记录即可将其内容复制到系统剪贴板
3. 在目标位置粘贴即可

### 记录操作

每条记录提供以下操作按钮：

- **复制**: 将内容复制到剪贴板
- **编辑**: 修改记录内容
- **删除**: 删除该记录
- **二维码**: 生成二维码

## 清空历史记录

### 清空所有记录

1. 点击右上角的菜单按钮
2. 选择"清空历史记录"
3. 确认操作

### 删除单条记录

1. 在记录上点击删除按钮
2. 确认删除操作
