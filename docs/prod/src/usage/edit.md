# 内容编辑

Ribopper 提供了方便的内容编辑功能，让您可以修改剪贴板历史记录中的内容。

## 打开编辑窗口

### 方法一：通过历史记录窗口

1. 打开剪贴板历史窗口
2. 找到要编辑的记录
3. 点击记录上的编辑按钮

### 方法二：通过快捷键

1. 在历史记录窗口中选中要编辑的记录
2. 使用编辑快捷键（默认可在设置中查看和修改）

## 编辑文本内容

### 文本编辑器

编辑窗口提供文本编辑器，支持：

- 基本文本编辑
- 多行文本编辑
- 文本格式化

### 保存编辑

1. 编辑完成后，点击"保存"按钮
2. 内容将更新到剪贴板历史记录
3. 更新后的内容会自动复制到系统剪贴板

### 取消编辑

点击"取消"按钮可以放弃当前的编辑，保留原始内容。

## 编辑限制

### 支持编辑的类型

- **文本内容**: 完全支持编辑
- **代码内容**: 支持编辑，保留格式
- **文件路径**: 支持编辑路径文本

### 不支持编辑的类型

- **二进制内容**: 暂不支持编辑

## 注意事项

- 编辑后的内容会覆盖原记录
- 编辑操作会更新记录的时间戳
- 编辑后的内容会自动复制到系统剪贴板
- 不支持编辑的记录类型，编辑按钮将被禁用
