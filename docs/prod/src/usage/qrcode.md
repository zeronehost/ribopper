# 二维码生成

Ribopper 提供了便捷的二维码生成功能，可以将文本内容快速转换为二维码。

## 打开二维码窗口

### 方法一：通过历史记录窗口

1. 打开剪贴板历史窗口
2. 找到文本类型的记录
3. 点击记录上的二维码按钮

### 方法二：通过快捷键

1. 在历史记录窗口中选中文本类型的记录
2. 使用二维码快捷键（默认可在设置中查看和修改）

## 二维码显示

### 实时预览

二维码窗口会实时显示生成的二维码

### 二维码特性

- **自动调整**: 根据文本长度自动调整二维码大小
- **高清晰度**: 生成高清晰度的二维码图像
- **快速生成**: 即时生成，无需等待

## 使用二维码

### 扫描二维码

使用手机或其他设备的扫描功能扫描生成的二维码：

1. 打开设备的相机或二维码扫描应用
2. 对准二维码图像
3. 扫描成功后即可获取文本内容

### 复制二维码

1. 在二维码窗口中点击复制按钮
2. 二维码图像将复制到剪贴板
3. 可以粘贴到其他应用中使用

## 支持的文本类型

### 完全支持

- 普通文本
- URL 链接
- 代码片段
- 配置信息

### 注意事项

- 文本过长可能导致二维码过于复杂
- 建议文本长度不超过 300 个字符
- 特殊字符可能影响二维码识别率
