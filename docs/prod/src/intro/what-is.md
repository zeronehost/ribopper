# 什么是 Ribopper

Ribopper 是一个基于 Tauri 和 Vue.js 的跨平台剪贴板管理工具，支持文本和文件格式，提供系统托盘、快捷键操作和数据持久化功能。

## 设计目标

Ribopper 的设计目标是：

1. **简化剪贴板管理** - 提供直观的界面，轻松管理剪贴板历史记录
2. **跨平台支持** - 支持 Windows、macOS 和 Linux 等主流操作系统
3. **高效性能** - 优化的算法确保快速处理剪贴板数据
4. **数据安全** - 本地存储，保护您的隐私和数据安全

## 适用场景

Ribopper 适用于以下场景：

- 需要频繁复制粘贴文本或文件的用户
- 需要追溯剪贴板历史记录的用户
- 需要快速访问之前复制内容的用户
- 需要对剪贴板内容进行编辑和管理的用户

## 技术架构

Ribopper 采用现代技术栈构建：

- **前端**: Vue.js 3, TypeScript, Vite
- **后端**: Rust, Tauri
- **数据库**: SQLite
- **UI组件**: Sober UI
- **状态管理**: Pinia

## 核心特性

- 多类型支持：管理文本和文件类型的剪贴板内容
- 系统托盘集成：左键显示剪贴板历史，右键菜单提供快速操作
- 数据持久化：使用 SQLite 数据库存储剪贴板历史记录
- 快捷键操作：自定义全局快捷键，支持编辑、删除、切换记录等操作
- 二维码生成：将文本内容转换为二维码
- 收藏功能：标记重要剪贴板内容
- 主题切换：支持浅色、深色和自动主题模式
