# 安装指南

本指南将帮助您在不同平台上安装 ribo-pper 剪贴板管理工具。

## 系统要求

在安装 ribo-pper 之前，请确保您的系统满足以下要求：

- **操作系统**: Windows 10+, macOS 10.15+, 或主流 Linux 发行版
- **内存**: 至少 4GB RAM
- **磁盘空间**: 至少 100MB 可用空间

## 安装方法

### 从预构建包安装（推荐）

从项目的 [Releases](https://github.com/zeronehost/ribopper/releases) 页面下载适合您操作系统的安装包：

- Windows: 下载 `.exe` 安装程序
- macOS: 下载 `.dmg` 镜像文件
- Linux: 下载 `.AppImage` 或对应的发行版包

下载后，按照常规方式安装应用程序即可。

## 卸载

### Windows
通过"控制面板" > "程序和功能"卸载 Ribopper

### macOS
将应用程序拖到废纸篓

### Linux
使用包管理器卸载，或直接删除安装的文件
