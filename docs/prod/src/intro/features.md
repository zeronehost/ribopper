# 功能特性

ribo-pper 提供了一系列强大的功能，帮助您高效管理剪贴板内容。

## 核心功能

### 1. 多类型剪贴板支持
- **文本内容** - 支持普通文本、富文本、代码等多种格式
- **文件内容** - 支持文件路径复制和文件管理

### 2. 系统托盘集成
- **左键点击** - 快速打开剪贴板历史窗口
- **右键菜单** - 提供快速操作选项
- **状态指示** - 显示应用运行状态

### 3. 数据持久化
- **SQLite 数据库** - 本地存储剪贴板历史记录
- **自动保存** - 所有剪贴板内容自动保存
- **历史记录管理** - 可配置历史记录数量限制

### 4. 快捷键操作
- **全局快捷键** - 自定义全局快捷键快速访问
- **编辑快捷键** - 快速编辑选中的剪贴板内容
- **删除快捷键** - 快速删除不需要的记录
- **切换快捷键** - 在剪贴板记录之间快速切换

### 5. 二维码生成
- **文本转二维码** - 将文本内容快速转换为二维码
- **即时预览** - 实时预览生成的二维码
- **一键复制** - 方便分享和使用

### 6. 收藏功能
- **标记重要内容** - 标记重要的剪贴板内容
- **快速访问** - 快速访问收藏的内容
- **分类管理** - 对收藏内容进行分类管理

### 7. 主题切换
- **浅色主题** - 适合白天使用的明亮主题
- **深色主题** - 适合夜间使用的暗色主题
- **自动主题** - 根据系统设置自动切换主题

## 高级特性

### 搜索功能
- **全文搜索** - 快速搜索剪贴板历史记录
- **过滤选项** - 按类型、时间等条件过滤结果
- **高亮显示** - 搜索结果高亮显示

### 数据管理
- **批量操作** - 批量删除、导出剪贴板记录
- **数据导出** - 支持导出剪贴板历史记录
- **数据备份** - 支持数据备份和恢复

### 性能优化
- **内存优化** - 高效管理内存使用
- **快速加载** - 快速加载历史记录
- **流畅体验** - 优化的界面交互体验
