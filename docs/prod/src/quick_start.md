# 快速开始

本指南将帮助您快速上手 Ribopper 剪贴板管理工具。

## 初次使用

### 1. 启动应用

安装完成后，启动 Ribopper。应用程序会在系统托盘显示图标，表示应用正在运行。

### 2. 打开剪贴板历史

- **方法一**: 左键点击系统托盘图标
- **方法二**: 使用默认快捷键（可在设置中自定义）

### 3. 查看剪贴板记录

在打开的窗口中，您可以看到所有剪贴板历史记录，包括：
- 文本内容
- 文件路径

## 基本操作

### 复制内容

1. 复制任意内容（文本或文件）
2. 内容会自动保存到剪贴板历史
3. 在剪贴板历史窗口中查看新添加的记录

### 使用剪贴板内容

1. 在剪贴板历史窗口中找到需要的记录
2. 点击记录即可将其复制到系统剪贴板
3. 在目标位置粘贴即可

### 编辑内容

1. 在剪贴板历史窗口中找到要编辑的记录
2. 点击编辑按钮
3. 修改内容后保存

### 生成二维码

1. 选择文本类型的剪贴板记录
2. 点击二维码按钮
3. 查看并使用生成的二维码

## 访问设置

右键点击系统托盘图标，选择"设置"即可打开设置窗口，在设置中您可以：

- 调整历史记录数量限制
- 配置开机启动
- 切换主题
- 自定义快捷键
- 配置操作规则
