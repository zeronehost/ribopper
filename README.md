# ribo-pper

一个基于 Tauri 和 Vue.js 的跨平台剪贴板管理工具，支持文本、图片和文件管理，提供系统托盘、快捷键操作和数据持久化功能。

## 功能特性

- **多类型支持**：管理文本、图片和文件类型的剪贴板内容
- **系统托盘集成**：左键显示剪贴板历史，右键菜单提供快速操作
- **数据持久化**：使用 SQLite 数据库存储剪贴板历史记录
- **快捷键操作**：自定义全局快捷键，支持编辑、删除、切换记录等操作
- **二维码生成**：将文本内容转换为二维码
- **收藏功能**：标记重要剪贴板内容
- **主题切换**：支持浅色、深色和自动主题模式

## 安装步骤

### 前提条件
- Node.js (v24+) 和 pnpm
- Rust 开发环境
- Tauri 依赖 (参考 [Tauri 文档](https://tauri.app/zh-cn/docs/getting-started/prerequisites))

### 安装依赖
```bash
pnpm install
```

## 开发与构建

### 启动开发环境
```bash
cargo tauri dev
```

### 构建生产版本
```bash
cargo tauri build
```

## 使用方法

### 基本操作
1. 系统托盘左键点击打开主窗口
2. 查看剪贴板历史记录和收藏内容
3. 点击列表项显示操作按钮：编辑、删除、生成二维码等
4. 使用快捷键快速操作（可在设置中自定义）

## 配置说明
在设置窗口中可以配置：
- 历史记录数量限制
- 自动启动选项
- 主题偏好
- 快捷键自定义

## 技术栈
- **前端**: Vue.js 3, TypeScript, Vite
- **后端**: Rust, Tauri
- **数据库**: SQLite
- **UI组件**: Sober UI
- **状态管理**: Pinia
